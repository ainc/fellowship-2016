Core Value Training

This game illustrates the relevance of the core values, and how they can be applied to your life. By completing various tasks, you will learn the meaning of each core value and why they are essential if you hope to pursue your definition of awesome!

Why We Use the Game

This game is designed to introduce you to the Core Values that Awesome Inc stands for. You should already know them by now: be good, be excellent, be a friend, and be you. Team Alpha members and employees are trained on these four values and encouraged to not only practice them at Awesome Inc, but to adopt them in their own lives.

What You'll Be Doing

You will be tasked with building a house. To gather the materials for your house, you will have to complete a series of challenges. These tasks include a memory game, a matching game, a snake game, and a minesweeping game. Each task represents a core value, and you must complete the challenges consecutively. As you build your house, you will learn more about the core values and what it takes to be Awesome!
